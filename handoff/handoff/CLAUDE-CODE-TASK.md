# Recall — редизайн всего приложения («Nocturne»)

Задача для Claude Code в VS Code, проект `recall-app`.
Визуальный референс — прототипы в Omelette: `Recall Вход.dc.html` (вход) и
`Recall Приложение.dc.html` (все экраны, утверждены пользователем).
Работать поэтапно, после каждого этапа — `npm run build` + смоук.

## Дизайн-система «Nocturne» (общая для всех экранов)

Токены и keyframes — `handoff/index.css.additions.css` (дописать в `src/index.css`).
Шрифт: **Inter** (400/500/600, Google Fonts, подключить в `index.html`).
`--night-font: 'Inter', ui-sans-serif, system-ui, sans-serif`.

- Фон `--night-bg` #161826 (тёмный сине-серый, не чёрный); текст `--night-text` #e9e9ed;
  поверхности карточек `--night-surface` #1e2030; поля ввода `--night-input` #262939;
  один акцент `--night-accent` #9184d9 (линии, рамки, мелкие метки — не заливка площадей);
  для мелкого акцентного текста `--night-accent-text` #b9b0e8.
- Заголовки weight 500, `tracking-tight`; иерархия размером и воздухом.
- Радиусы: 12px поля/кнопки, 16px карточки-строки, 20–24px крупные панели.
- Карточка-строка (основной паттерн списков): `flex items-center gap-3.5 rounded-2xl
  border border-white/[0.08] bg-[var(--night-surface)] px-4 py-3.5` + иконка в
  квадрате 40px `rounded-xl` (активная: bg accent-900 тон #2b2450, цвет #cfc8f0).
- Иконки: **Phosphor** (`@phosphor-icons/react`, вес regular; активная вкладка — fill).
  Заменяет самодельные `components/icons.tsx` постепенно, не ломая старые экраны.
- Главная CTA: светлая заливка `--night-text` с тёмным текстом ИЛИ акцентная
  outline-кнопка (`border-accent` + полупрозрачная заливка 14–22%) — как в прототипе.
- Анимации: `fade-up` со стаггером (delay 60–90ms на элемент), `lift` на ховер/тап,
  пружина `cubic-bezier(.22,1,.36,1)`; уважать `prefers-reduced-motion` (уже глобально).
- Тема: единая тёмная (светлую убрать с этих экранов; `color-scheme: dark`).

6. **Слово дня**: маленькая карточка — лампочка, «СЛОВО ДНЯ», `{front} — {back}`,

## Этап 4 — Онбординг (`/onboarding`, features/onboarding/OnboardingFlow.tsx)

После **успешной регистрации** (и только для нового пользователя — например, если ни
одной активности и уровень не задан) редирект на `/onboarding`. 3 шага, сверху
прогресс-полоски (3 сегмента, пройденные — accent):
1. «Что будем учить?» — две большие карточки EN / ES (квадрат с «EN»/«ES» вместо флагов);
   выбор пишет `recall.lang`.
2. «Определим уровень» — существующий PlacementTest встроить/переиспользовать
   (для EN пока: выбор уровня вручную A2–C1 карточками, теста EN нет); есть «Пропустить тест».
3. «Твой план готов» — seal-check иконка (popIn), «Уровень: … · ~15 минут в день»,
   3 строки плана (колода/чтение/речь, стаггер), кнопка «Начать первое занятие» →
   `/` + конфетти.

## Этап 5 — Экран прогресса (`/progress`, features/progress/ProgressPage.tsx)

Не в нижней навигации; вход: аватар в шапке и «Мой прогресс» в стрик-герое. Назад — caret.
- «Эта неделя»: столбчатый график 7 дней (высота = минуты или количество активностей
  из activity_log; анимация barGrow со стаггером), подпись «N минут · M дней».
- Сетка 2×2 метрик: слов изучено (cards с review_state state='review'), точность
  повторений (% good из последних N ревью — если считать сложно, доля reps без lapses),
  лучшая серия (max по activity_log), к повторению завтра (due < завтра+1).
- Кнопка «Выйти из аккаунта» (перенесена с Главной).
