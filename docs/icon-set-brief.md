# ТЗ на фирменный набор иконок Recall (для Claude Design)

Цель: единый **векторный** icon-набор, который заменит Phosphor во всём
приложении Recall (тёмная тема «Nocturne», шрифт Onest, акцент #9184d9).
Набор должен читаться как одна система — это важнее, чем красота каждой иконки
по отдельности.

## Жёсткие правила стиля (соблюдать для КАЖДОЙ иконки)
- **Формат:** SVG, отдельный файл на иконку. НЕ PNG, НЕ растр.
- **Канва:** `viewBox="0 0 24 24"`, вся графика в пределах поля 20×20 (по 2px
  воздуха с каждой стороны — оптический паддинг).
- **Стиль:** контурный (outline), НЕ заливка. Толщина линии **ровно 1.75px**,
  одинаковая у всех. `stroke-linecap="round"`, `stroke-linejoin="round"`.
- **Цвет:** только `stroke="currentColor"`, никакого фиксированного цвета и
  никаких градиентов. Цвет задаётся через CSS (тема + акцент). Внутри SVG
  не должно быть атрибутов `fill` кроме `fill="none"` на путях.
- **Сетка и метафоры:** единая геометрия — одинаковые радиусы скругления углов
  (~2px), одинаковый угол «плеча» у стрелок, один визуальный язык (гуманистичный
  геометрический, под шрифт Onest — мягкие, но чёткие формы).
- **Оптический вес:** все иконки должны выглядеть одинаково «плотными» в размере
  24px. Проверять набором рядом, а не поштучно.
- **Именование файлов:** kebab-case по смыслу (см. список): `speaker.svg`,
  `mic.svg`, `flame.svg`…
- Для активного состояния (выбранная вкладка) — отдельный **filled**-вариант
  тем же контуром + мягкая заливка: суффикс `-fill` (`mic-fill.svg`).

## Полный список иконок (≈40)

**Навигация (нижняя панель):**
- `home` — главная
- `practice` — тренировка (гантель/штанга)
- `study` — учёба (стопка книг)
- `speech` — речь (микрофон)
- `dialog` — диалог (облачко чата)
(для каждой — обычный и `-fill` вариант, fill для активной вкладки)

**Действия и статусы:**
- `speaker` — озвучить (динамик со звуком)
- `speaker-slow` — медленная озвучка (динамик + спидометр/черепаха-намёк)
- `mic`, `stop`, `spinner` — запись/стоп/загрузка
- `check` — верно/готово (галочка)
- `arrow-right` — «дальше», `arrow-up` — «тянись выше»
- `refresh` — повторить/ещё раз (круговая стрелка)
- `search` — поиск (лупа), `plus` — добавить
- `hint` — подсказка (лампочка)
- `send` — отправить (бумажный самолётик)
- `back` — назад (шеврон влево)
- `gear` — настройки, `sign-out` — выход
- `chart` — прогресс (график вверх)
- `flame` — стрик (огонёк)
- `sparkle` — «всё готово» (искра/блеск)
- `trophy` — отличный результат, `thumbs-up` — хороший результат
- `badge-check` — проверено (галочка в «печати»/розетке)
- `warning` — предупреждение (треугольник)

**Практика — плитки игр:**
- `meaning` — значение (звенья/связь)
- `gap` — пропуск в предложении (открытая книга)
- `translate` — перевод (буквы Аа)
- `headphones` — аудирование
- `timer` — спринт (таймер)
- `keyboard` — диктант
- `puzzle` — собери фразу
- `mcq` — выбери форму (чек-бокс)
- `pencil` — впиши слово
- `rows` — порядок слов (строки)
- `shuffle` — глаголы (перемешать)

**Преподаватель:**
- `teacher` — преподаватель (доска/учитель)
- `students` — ученицы (люди)
- `materials` — материалы (лист с пером)
- `graduation` — выпуск (шапочка)
- `printer` — печать

## Как отдать результат
Идеально — один запрос «сгенерируй ВЕСЬ набор в едином стиле по этим правилам,
покажи все 40 иконок сеткой 24px рядом друг с другом», потом сверка
консистентности, потом выдача каждой иконки отдельным SVG. Поштучные
независимые запросы дадут разнобой — так не делать.

## Интеграция (со стороны кода)
Иконки заведём в один модуль `src/components/icons.tsx` (семантические имена →
SVG-компоненты). Тогда замена Phosphor → фирменный набор = правки одного файла.
