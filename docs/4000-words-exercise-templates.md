# «4000 Essential English Words» — шаблоны упражнений (2026-07-23)

> Задача В захода 11+12: подтвердить однотипность упражнений серии и описать
> шаблоны для будущего генератора. Проверено глазами по страницам томов 1 и 5
> (юниты разных уровней) — структура идентична во всей серии.
> САМИ упражнения книги НЕ переносим (копирайт) — только формат.

## Структура юнита (одинакова во всех 6 томах)

1. **Word List** — ~10 целевых слов: слово + транскрипция + определение EN +
   пример-предложение (это извлекаем в паки «База уровня»).
2. **Exercises** — 3-5 однотипных блоков (см. шаблоны).
3. **Story** — текст ~200-300 слов, где все целевые слова выделены жирным.
4. **Reading Comprehension** — вопросы по тексту.

## Шаблоны упражнений (для нашего генератора)

### Шаблон A — Word bank fill-in
«Write the word from the word bank that best fits each sentence»: банк из
5-10 целевых слов + предложения с пропуском. У нас уже есть аналог — тип
`fill` с подсказкой-банком или `mcq` с 4 вариантами из банка.
Генератор: на каждое слово пака — предложение-контекст с пропуском, банк = слова юнита.

### Шаблон C — Перифраз → слово
«Write a word that is similar in meaning to the underlined part»: предложение,
в котором подчёркнуто ОПРЕДЕЛЕНИЕ слова («important sea officer that controls
military ships» → admiral). Отличный формат: тренирует значение, а не форму.
Генератор: берём определение из пака, встраиваем в предложение, ответ — слово.
У нас реализуемо как `fill` (напечатать слово) или `mcq`.

### Шаблон B/E — Выбор значения / понимание текста
«Choose the right word» (mcq по значению) и вопросы по тексту юнита — это
ровно наши mcq comprehension в материалах преподавателя.

## Вывод

Новых типов упражнений НЕ требуется: A → fill/mcq, C → fill по определению,
B/E → mcq. Будущий генератор «упражнения по паку слов» (в Практику или в
материалы преподавателя) может собирать раунд из шаблонов A и C по любому
паку «База уровня» — все данные (слово, определение EN, пример) в паках есть.
